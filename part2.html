Cheat Sheet for HTML tags

A

<a>
Defines a hyperlink
href = defines a hyperlink
hreflang= specifies the language of the document in the link
media= specifies what media the linked document is optimized for
ping = a list of URLs to be notified if the user follows the hyperlink
rel = soecifies the relationship between the current document and linked document
target = specifies where to open the linked document
type = specifies the media tyoe of the linked document

<abbr>
Define an abbreviation

<address>
Defines an address element

<area>
Defines an area inside an image map
alt = defines a alternate text
coords= defines the coordinates of an area in an image map
href = defines a hyperlink
hreflang= specifies the language of the document in the link
media= specifies what media the linked document is optimized for
ping = a list of URLs to be notified if the user follows the hyperlink
rel = soecifies the relationship between the current document and linked document
shape = specifies the size, shape, and placement of a link
target = specifies where to open the linked document

<audio>
Defines sound and audio streams
User can play/pause and modify the volume

<aside>
Defines content aside from the content it is placed in

<article>
Defines independent and self contained content

B

<br/>
Define a single line break

<b>
Defines bold text without any extra importance

<body>
Defines a document's body

<big>
Defines bigger

D

<del>
Defines text that has been deleted from a document

<div>
Defines a division or section in the HTML document
style= defines the design or elements associated with that section
align= defines the placement of the section

<datalist>
Defines a specific list that has pre-defined options for an <input> element
id

<!Doctype html>
Defines the document type

E

<em>
Defines emphasized text in the document

F

<footer>
Defines the footer for the document or section

<form>
Defines the creation of a HTML form for user input
Includes elements such as:
<input>= elemet included into the form
<textarea>= text input control
<button>= the button belonging to certain elements apart of the form
<select>= specifies the form the drop-down list belongs to
<option> = is what the browser displays as a drop-down list
<fieldset> = is used to group related elements
<label> = defines a label for many elements
<output> =defines elements outside of a form
<optgroup> = groups related options in a drop-down list

H

<html>
Defines notifying the browser that this is an HTML document

<head>
Defines a container for metadata(data about the HTML document)

<h1>,<h2>,<h3>,<h4>,<h5>,<h6>
Defines the different sizes for HTML headings

<hr>
Defines a thematic break in the page
width= specifies the width of a horizontal line

<header>
Defines container for introductory content or a set of navigational links

I

<i>
Defines content to be displayed in italics

<input>/<input type>
Defines a field where the user can insert data
align = defines the placement of what is inputed
height = specifies the height of the input
mas = specifies the maximum value for input
min = specifies the minimum value for input
src = specifies the URL of the image to use as a submit button
value = specifies the value of the input
radio = specifies a radio button needs to be displayed
checkbox = specifies a checkbox needs to be displayed

<ins>
Defines a text that has been inserted into the document

<img>
Defines a image in the HTML page
align=defines the placement of image
height= defines the height of the image
src = specifies the source of the image
width= defines the width of the image

<li>
Define a list item

N

<nav>
Define a set of navigation links

O

<ol>
Define an ordered list

P

<p>
Defines a paragraph
align

<progress>

<sub>
Defines subscript text ( where half the character is below the normal line)

<sup>
Defines superscript text (where half the character is above the normal line)

<section>
Defines a section in the document( chapters, headers, footers)

<span>
Define grouping inline-elements in a document

<script>
Define a client side script such as JavaScript

async= specifies the script is done asynchronously with the rest of the page

<strong>
Defines important text

<tr>

<th>

<td>
